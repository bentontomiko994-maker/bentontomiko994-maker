## Featured Project

### [Douban Archive](https://github.com/bentontomiko994-maker/douban-archive)

一个本地优先的豆瓣个人数据归档插件。

它可以帮助用户把自己在豆瓣上的书影音、广播、日记和评价记录导出到本地，支持 JSON、Excel CSV 和 Markdown。

这个项目的核心想法很简单：

> 把你的数字记忆从平台里带回本地。

未来这些导出的数据也可以成为 AI 更理解你的基础：你喜欢什么、讨厌什么、长期关注什么、兴趣如何变化。
